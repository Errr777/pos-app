# Invoice Feature Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a persistent A4 invoice (INV-YYYYMMDD-XXXX) to sale detail, installment plan, and purchase order pages, each opening as a popup print window with no app chrome.

**Architecture:** Four migrations add `invoice_number` + `invoice_issued_at` to three tables plus a new `invoice_sequences` counter table. A helper class generates race-safe numbers. Three controller methods generate/reuse numbers and render bare Inertia print pages. Two React print pages (no AppLayout) render the A4 invoice layout and auto-trigger `window.print()`.

**Tech Stack:** Laravel 12, Inertia.js v2, React 19, TypeScript, Tailwind CSS v4, SQLite (dev) / MariaDB (prod)

```php
<?php

namespace App\Helpers;

use Illuminate\Support\Carbon;
use Illuminate\Support\Facades\DB;

class InvoiceNumber
{
    public static function generate(): string
    {
        $today = Carbon::today()->toDateString();

        return DB::transaction(function () use ($today) {
            $row = DB::table('invoice_sequences')
                ->where('date', $today)
                ->lockForUpdate()
                ->first();

            if ($row) {
                $sequence = $row->sequence + 1;
                DB::table('invoice_sequences')
                    ->where('date', $today)
                    ->update(['sequence' => $sequence, 'updated_at' => now()]);
            } else {
                $sequence = 1;
                DB::table('invoice_sequences')->insert([
                    'date'       => $today,
                    'sequence'   => $sequence,
                    'created_at' => now(),
                    'updated_at' => now(),
                ]);
            }

            return 'INV-'
                . str_replace('-', '', $today)
                . '-'
                . str_pad($sequence, 4, '0', STR_PAD_LEFT);
        });
    }
}
```
